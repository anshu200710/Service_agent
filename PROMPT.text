THE COMPLETE CALL JOURNEY:
STEP 1: Call Starts (handleInitialCall)

Customer receives the call
System gets the call ID (CallSid) from Twilio
System checks if customer info exists (from memory store)
If new call → Creates a new Call record in database with:

Phone number
Customer name
Machine details (model, serial number)
Service type
Due date


Records the start time
Agent speaks in Hindi: "Hello! Can I speak with [Name]? Your [Machine Model] needs service."
Waits for customer to respond


STEP 2: Customer Speaks (handleUserInput - Part 1)

Customer responds
System captures the speech and converts to text
If customer says nothing → asks them to repeat
If customer speaks → continues to next step


STEP 3: AI Processes Response

System sends all previous messages + new customer message to AI (Groq)
AI reads the conversation history
AI understands what customer said in context
AI generates a response in Hindi


STEP 4: Check for Booking Intent

System looks at AI response for booking decision
AI might say one of 3 things:

✅ "Confirmed" → Customer agrees to book a date
⏭️ "Already Done" → Customer says service was already done
📅 "Rescheduled" → Customer wants to reschedule




STEP 5A: If Service is CONFIRMED

System extracts the service date
Finds the nearest service center/branch in customer's city
Saves to database:

Confirmed date
Branch name & location
Branch code


Marks outcome as "confirmed"
Agent thanks customer in Hindi and hangs up
Call is marked as "completed"


STEP 5B: If Service Already DONE

System records where it was done (location)
Marks outcome as "already_done"
Agent says thank you and hangs up
Call is marked as "completed"


STEP 5C: If No Booking Yet

AI response continues conversation
System saves AI message
Agent speaks the AI response
Waits for next customer response
Turn counter increases by 1


STEP 6: Safety Limit (Infinite Loop Protection)

If conversation reaches 12 turns (customer + agent = 12 exchanges)
System automatically:

Stops the call
Says polite goodbye in Hindi
Hangs up
Marks call as completed
Records total duration
